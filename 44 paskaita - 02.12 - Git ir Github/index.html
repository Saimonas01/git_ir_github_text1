Git ir Github

Git'as - kodo versijavimo programa.
GitHub'as - versijuoto kodo talpykla internete.

Git komandos

  • init - sukurti git repozitorija dabartiniame aplankale
  • status - pateikia git repozitorijos dabartine informacija
  • add - prideda nurodytus failus pries commit'inimui paruostu failu (stage'ina)
  • rm --cached || restore --staged - isimanurodytus failus ir stage'inimo zonos (unstage'ina)
  • commit -m "zinute apie commit'a" - sukuria kodo versija is stage'intu failu ir prideda zinute apie pakitimus (zinute turetu buti trumpa ir aiski)
  • restore - sugrazina nurodyta/us i pradine busena (jeigu tie failai nebuvo add'inti i stage'ingo zona)
  • push - nusiuncia lokalios repozitorijos su'commit'intus i isorine repozitorija
  • pull - atsiuncia isrorines repozitorijos pakitimus i lokalia repozitorija
